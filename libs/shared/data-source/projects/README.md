# Projects Data Source

This library provides GraphQL-based data access for projects and categories.

## Services

- `ProjectService` - Handles all project-related GraphQL operations
- `CategoryService` - Handles all category-related GraphQL operations

## GraphQL Operations

All GraphQL operations are defined in the `graphql/` directory:
- `projects.gql` - Project queries and mutations
- `categories.gql` - Category queries and mutations

## Types

TypeScript interfaces for projects and categories are defined in `types.ts`.
