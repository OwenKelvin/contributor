# Contributions Management Components

This library contains shared components for the contributions management feature.

## Components

### PaymentStatusBadgeComponent
Displays a colored badge based on payment status (pending, paid, failed, refunded).

### UserAutocompleteComponent
Provides user search functionality with debounce and integrates with reactive forms.

### ContributionFiltersComponent
Provides filtering capabilities for contributions including status, project, date range, and amount range.

### PaginationComponent
Handles pagination for contribution lists with page size selection and navigation controls.

## Usage

```typescript
import {
  PaymentStatusBadgeComponent,
  UserAutocompleteComponent,
  ContributionFiltersComponent,
  PaginationComponent
} from '@nyots/admin/pages/contributions';
```
